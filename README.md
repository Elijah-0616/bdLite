## 使用方法
### 浏览器访问以下地址
>　https://search.elijah16t.ml

### 本地运行
克隆项目到本地
> 如果你已经fork此项目，请将以下命令中的 Elijah-0616 替换为你的 Github 用户名。

```bash
git clone --recursive https://github.com/Elijah-0616/bdLite.git
```
用浏览器打开index.html文件
